# Handstand Tracker — Web Frontend

Static SPA frontend for the Handstand Training Progress Tracker.

## Setup

1. Update `API_URL` in `public/index.html` to point at your deployed API
2. Push to GitHub
3. Render → New → Blueprint → connect this repo

## Local

```bash
npm install
npm run dev
```

Expects the API running at `http://localhost:4000`.
